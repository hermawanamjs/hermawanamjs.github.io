Macam-Macam Bahasa Pemrograman

JavaScript

JavaScript, sering disingkat JS, adalah bahasa pemrograman berorientasi objek tingkat tinggi, tepat waktu yang dikompilasi, yang sesuai dengan spesifikasi ECMAScript.

Rilis : Tahun 1995;

Rilis Stabil : ECMAScript 2019/Juni 2019;

Pengembangan : Ecma International (Sekarang)

Dibuat Oleh : Brendan Eich

Lebih Lanjut

PHP

PHP adalah bahasa pemrograman tujuan umum yang awalnya dirancang untuk pengembangan web. Awalnya dibuat oleh Rasmus Lerdorf pada tahun 1994; implementasi referensi PHP sekarang diproduksi oleh The PHP Group.

Rilis : Tahun 1994;

Rilis Stabil : PHP 7.4.1/Desember 2019;

Pengembangan : Zend Technologies (Sekarang)

Dibuat Oleh : Rasmus Lerdorf

Lebih Lanjut

C++

C++ adalah bahasa pemrograman untuk tujuan umum yang dibuat oleh Bjarne Stroustrup sebagai perpanjangan dari bahasa pemrograman C, atau "C dengan Kelas". C++ memiliki fitur berorientasi objek, generik, dan manipulasi memori.

Rilis : Tahun 1985;

Rilis Stabil : C++17/Desember 2017

Pengembangan : ISO/IEC JTC1 (Sekarang)

Dibuat Oleh : Bjarne Stroustrup

Lebih Lanjut

Python

Python adalah bahasa pemrograman tujuan umum tingkat tinggi yang ditafsirkan. Dibuat oleh Guido van Rossum dan filosofi desain Python menekankan keterbacaan kode dengan penggunaan spasi spasi yang signifikan.

Rilis : Tahun 1991;

Rilis Stabil : Python 3.8.0/Oktober 2019;

Pengembangan : Python Software Foundation (Sekarang)

Dibuat Oleh : Guido van Rossum

Lebih Lanjut

Java

Java adalah bahasa pemrograman tujuan umum tingkat tinggi yang dibuat oleh James Gosling di Sun Microsystems, yang kemudian diakuisisi oleh Oracle Corporation, yang menyediakan sistem untuk mengembangkan perangkat lunak aplikasi dan menyebarkannya dalam lingkungan komputasi lintas platform.

Rilis : Tahun 1991;

Rilis Stabil : Java 11/September 2018;

Pengembangan : Oracle Corporation (Sekarang)

Dibuat Oleh : James Gosling

Lebih Lanjut

C#

C # adalah bahasa pemrograman multi-paradigma tujuan umum yang mencakup pengetikan yang kuat, cakupan leksikal, imperatif, deklaratif, fungsional, generik, berorientasi objek, dan disiplin pemrograman berorientasi komponen.

Rilis : Tahun 2000;

Rilis Stabil : C# 8.0/September 2019;

Pengembangan : Microsoft Corporation

Dibuat Oleh : Anders Hejlsberg
